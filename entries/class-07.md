### Tables
#### HTML pgs 126-145

`<table>` written row by row
  - `<tr>` start of each row
    - `<td>` each cell of a table
      - `colspan="2"` stretches a cell across columns
      - `rowspan="2"` can be used on th or td to span cells
  - `<th>` table heading

`<thead>`
`<tbody>`
`<tfoot>`
